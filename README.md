# UAV Image Quality–Driven Control using Reinforcement Learning

## Overview
This project proposes a perception-driven control framework for UAVs in precision agriculture.

The UAV dynamically adjusts its altitude based on real-time image quality assessment (IQA) using reinforcement learning.

## Key Components
- MM-IQA (Multi-Metric Image Quality Assessment)
- DDQN / PPO reinforcement learning
- UAV altitude control logic
- YOLO-based pest detection

## Results
- Stable altitude convergence
- Improved image quality consistency
- Real-time adaptive control
